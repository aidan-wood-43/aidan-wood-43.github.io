Connecting Power BI to a Post API with Power Query

Recently I had to configure a Power BI data source to pull data from a POST api. The ability to construct a POST request isn’t natively available as part of any default or third party connector on Power BI, so I had to construct the query in Power BI. The process isn’t too involved and hopefully this helps anyone facing a similar problem. As a Power BI / Power Query / BI / general novice, I made extensive use of an excellent blog post by Chris Webb - check it out for further context on the solution.
